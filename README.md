OVERVIEW
--------
FitLife Pro is a full-featured fitness platform that combines
WordPress and Shopify to deliver a seamless shopping and content
experience. It includes a custom WordPress theme, WooCommerce
product management, Shopify Liquid theme sections, and a REST API.


FEATURES
--------
- Custom WordPress Theme
- WooCommerce Integration
- Custom Post Types
- Gutenberg Blocks
- Shopify Theme Sections
- Shopify API Integration
- REST API
- Responsive Design


TECHNOLOGIES
------------
CMS          : WordPress
E-commerce   : WooCommerce, Shopify
Templating   : Shopify Liquid
Backend      : PHP, REST API
Frontend     : JavaScript, CSS


INSTALLATION
------------
WordPress Theme:
  1. Copy fitlife-pro folder into wp-content/themes/
  2. Go to Appearance > Themes and activate FitLife Pro

Plugin:
  3. Copy fitlife-core into wp-content/plugins/
  4. Go to Plugins and activate FitLife Core

WooCommerce:
  5. Install and activate the WooCommerce plugin
  6. Run the WooCommerce setup wizard

Demo Content (optional):
  7. Import the demo content file for pre-populated pages and products
